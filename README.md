# YP-Spur (YamabicoProject-Spur)

## Purpose

YP-Spur is a mobile robot motion control software with coordinate frame based commands. 

## Getting Started

Learn to use YP-Spur at the [documentation pages](./doc/README.md).

## Install
```
$ mkdir build
$ cd build
$ cmake ..
$ make
$ sudo make install
```
